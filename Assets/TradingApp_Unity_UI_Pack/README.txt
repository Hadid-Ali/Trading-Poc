Trading Learning App — Unity UI Pack

Contents
- Screens/: 11 separate mobile screen mockups.
- Elements/Mascot/: Bull mascot crops for guidance and onboarding.
- Elements/Cards/: Reusable card and row components.
- Elements/Buttons/: Main action buttons and mini-game controls.
- Elements/Navigation/: Header and bottom navigation components.
- Elements/MiniGames/: Two mini-game playfield crops.

Unity Setup
1. Import PNG files into Assets/UI.
2. Set Texture Type to Sprite (2D and UI).
3. Use Canvas Scaler: Scale With Screen Size.
4. Recommended reference resolution: 1080 x 1920.
5. Rebuild interactive text, charts and buttons with Unity UI Toolkit or UGUI.

Important
These assets are concept-quality crops from a flattened mockup, not true layered source files.
